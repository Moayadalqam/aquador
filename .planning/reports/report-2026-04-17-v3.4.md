## What Was Done

Three coordinated scroll/loading/3D upgrades executed in parallel (3 builders, one wave), all shipped together:

### 1. WelcomeSplash — branded first-visit loading state
Replaces the 800ms blocker removed in v3.3 with a tasteful, premium pattern:
- **sessionStorage gate** (`aquador_visited`) — shown ONLY on first visit per session; return visits/refreshes see nothing
- **Max 400ms** normal, **150ms** on `prefers-reduced-motion`
- **Non-blocking** — fixed overlay at z-9999 with `pointer-events-none`, renders over SSR content so LCP is unaffected
- **Branded** — gold-gradient `AQUAD'OR` wordmark + animated hairline + "Scent of Luxury" eyebrow
- **Dynamic imported** via `dynamic(..., { ssr: false })` — zero server bundle impact
- **Accessible** — `role="presentation"`, `aria-hidden="true"`

File: `src/components/ui/WelcomeSplash.tsx`

### 2. Gold scroll progress bar
Subtle 2px gradient indicator fixed at top of viewport, spring-smoothed via framer-motion `useSpring`. Respects `prefers-reduced-motion` (returns null). Purely ornamental — `aria-hidden`, `pointer-events-none`.

Also added `useReducedMotion` guard to Hero's ambient glow breathing animation (minor a11y tuning surfaced during this pass).

File: `src/components/ui/ScrollProgress.tsx`

### 3. Scroll-driven 3D hero showcase (the "wow")
New full-viewport section between Hero and TrustBar — a procedural gold perfume bottle that rotates, scales, and floats based on scroll position through the section:

- **R3F scene** (`Hero3DScene.tsx`) — procedural low-poly bottle: capsule body + shoulder taper + neck + black cap + gold ring. 24 orbiting icosahedron particles for ambient depth.
- **Scroll-driven transforms** via framer-motion `useScroll` + `useTransform`:
  - Rotation: 0 → 2π through the section (full spin)
  - Scale: 0.75 → 1.1 → 0.9 (breathe)
  - Y position: -0.5 → 0.5 (drift up)
  - Subtle X-axis tilt for character
  - Continuous idle float on top via `useFrame`
- **Dynamic imported** via `dynamic(() => import('./Hero3DScene'), { ssr: false })` — bundle unchanged at 88.4KB
- **Device-gated**: `useDeviceCapabilities().supports3D` check — mobile/low-end gets a layered CSS-gradient fallback with gold atmospheric glow (still beautiful, no Three.js download)
- **Reduced-motion fallback**: same CSS-only treatment
- **Foreground copy**: "Crafted in Cyprus" eyebrow + "The Signature Collection" gold-gradient Playfair headline + subtitle, all whileInView-animated

Files: `src/components/home/Hero3DScroll.tsx` (wrapper + device check), `src/components/home/Hero3DScene.tsx` (R3F scene)
