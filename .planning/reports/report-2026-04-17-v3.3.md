## What Was Done

### Full-spectrum optimization pass
After v3.2 shipped, re-ran `/qualia-optimize` which surfaced 33 new findings (0 critical, 10 high, 15 medium, 8 low). Executed all 33 fixes in two parallel build waves (12 builders total), landed clean TSC + build, applied 3 Supabase migrations, deployed, verified.

### Security (4 HIGH)
- **Admin privilege escalation closed** — `/admin/settings` user creation moved to `/api/admin/users` with `super_admin` role gate, Zod validation, service_role writes, rate limiting. Client can no longer self-promote.
- **`site_visitors` DELETE RLS policy** — explicit policy added for defense-in-depth (migration applied).
- **Live chat session_secret** — new column + tightened UPDATE policy (migration applied). SELECT still open pending API-route-based auth.
- **Session details 24h TTL** — `/api/checkout/session-details` rejects stale sessions, preventing indefinite product/price exposure.

### Performance
- **LoadingScreen overlay removed** — eliminates 800-1300ms of perceived load time on every page. SSR content is already visible; overlay was legacy SPA pattern.
- **React.memo on ProductCard + CartItem** — prevents 100+ unnecessary re-renders when filters change in the shop grid.
- **Middleware narrowed to `/admin/:path*`** — removes UUID generation + header clone overhead from every `/api/*` request.
- **Heartbeat probabilistic cleanup** — DELETE now runs on ~1% of requests via `Math.random() < 0.01` + new `last_seen` index (migration applied).
- **AnimationBudgetProvider auto-stops** — RAF loop stops after 5s measurement instead of running indefinitely.
- **Supabase public client singleton** — cached at module scope; saves 387+ throwaway instances per build.
- **AI `ALL_KEYWORDS` IIFE hoist** — no longer recomputes the Set on every POST.
- **`searchProducts` `.limit(20)`** — scalability guard.
- **FeaturedProducts `priority={index < 3}`** — first 3 LCP images marked priority; Hero poster preload link added.
- **Poppins `display: "optional"`** — reduces FOUT/CLS from body font.

### A11y
- **CartDrawer Escape + focus trap + focus restore** — WCAG 2.1 SC 2.1.2 compliant.
- **Desktop nav dropdown keyboard-accessible** — added `group-focus-within` + aria-haspopup/aria-expanded.
- **Mobile menu Escape dismiss**.
- **Form aria-describedby + aria-invalid + role="alert"** on contact, checkout, reorder.
- **Blog pagination aria-current + aria-label**.

### UX polish
- **Shop empty state** upgraded to match category-page pattern (icon + heading + action).
- **Checkout success** — `Loader2` spinner during fetch (green check reserved for confirmed state). `useRef` guard against double-clear.
- **SwipeableProductGrid** hints now conditional on `isSwiping`.
- **`onError` state-tracked** on product images (no more DOM mutation breaking next/image).
- **`global-error.tsx`** uses Playfair + Poppins via Google Fonts link.

### Backend refactor
- **Product/category CRUD → server API routes** (`/api/admin/products`, `/api/admin/categories`) with `revalidatePath` to eliminate 30min storefront staleness after admin writes.
- **Admin `select('*')` → named columns** across 7 files (dashboard, settings, categories, products edit, customers, live-chat, ChatWidget).
- **CSP connect-src documented** — code comment explains openrouter.ai (AI assistant) + why api.resend.com/graph.facebook.com stay out (server-side only).
